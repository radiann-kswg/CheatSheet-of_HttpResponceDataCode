# 2026-07-16 進捗レポート

## 目的

GitHub Pages を使って、HTTPステータスコードのチートシートをWeb上で閲覧できるようにする。加えて、各ページ内のレスポンスサンプル（`http` コードブロック）を見やすく抽出表示する。

## 変更点の要約

- GitHub Pages 向けのデプロイワークフローを追加
- ルート配信の Web ビューアー（`index.html` + `assets`）を追加
- ステータスコード一覧の検索機能を追加
- 個別ページから `http` コードブロックを抽出し「レスポンスサンプル」として表示
- サンプルのコピー機能を追加
- README に公開手順とローカル確認手順を追記

## 影響範囲（編集したファイル）

- `.github/workflows/deploy-pages.yml`
- `index.html`
- `assets/app.css`
- `assets/app.js`
- `README.md`

## 未完了タスク
